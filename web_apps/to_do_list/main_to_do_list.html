To Do List

How to use the Web App!

  1. To add an item write the item in the text field below and press "Add to List" button.
  2. To edit the item after you have added it to the list, you double click on the text you want to change and write something new.
  3. To check off an item click on the check box to the left of the item and it will turn green to tell you, you have checked it off.
  4. To delete an item from the list you can press the delete button to the right of the list item.
